# DeepPhilosophy — 深哲

**东西方及世界哲学的综合知识平台**。探索 103 个哲学流派、599 位哲人、数百部经典著作。

---

## 功能

| 模块 | 说明 |
|------|------|
| 哲学掠影 | 博物馆级谱系图录，109 张星丛，按时代/地域纵横浏览 |
| 流派详情 | 概览、时间轴、辞海、金句画廊、著作列表、结语 |
| 哲人 | 599 位哲学家，含头像、生平、流派、著作 |
| 书籍 | 东西方哲学经典，支持 PDF/EPUB 在线阅读 |
| AI 问答 | DeepSeek 驱动的哲学问答，流式输出 |
| 游戏 | 答案之书、PHTI 哲学人格测试 |

---

| 指标 | 数量 |
|------|------|
